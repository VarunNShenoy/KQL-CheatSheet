# KQL Cheat Sheet

A practical Kusto Query Language (KQL) reference for Security Operations Center (SOC) analysts, threat hunters, and Microsoft Sentinel users.

## What is KQL?

Kusto Query Language (KQL) is used to query and analyze large volumes of log data in:

- Microsoft Sentinel
- Microsoft Defender XDR
- Azure Monitor
- Log Analytics Workspace
